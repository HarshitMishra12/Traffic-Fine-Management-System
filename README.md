# Smart Traffic Fine Management System

A digital e-challan management system built using Java, Spring Boot, MySQL, HTML, CSS, and JavaScript.

## Features

- Vehicle registration
- Traffic offense tracking
- Challan generation
- Fine management
- Real-time analytics dashboard
- REST API backend

## Tech Stack

Backend:
- Java
- Spring Boot

Frontend:
- HTML
- CSS
- JavaScript
- Chart.js

Database:
- MySQL

## Dashboard

Displays:
- Total vehicles
- Challans issued
- Fine collection
- Violation analytics
- Recent traffic violations

## How to Run

1. Start MySQL
2. Run Spring Boot application
3. Open:

http://localhost:8080/index.html
